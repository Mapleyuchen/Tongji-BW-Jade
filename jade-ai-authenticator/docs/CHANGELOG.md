# 更新记录

## v2.0.0

针对正式项目体验进行了增强：

- 实现真正的拖拽上传：页面阻止浏览器默认打开图片，拖入区域会出现上传遮罩。
- 实现多图上传：前端字段统一为 `images`，后端支持 `images` / `image` / `files`。
- 默认最多上传 6 张图片，支持继续添加、删除、清空、重复图片跳过。
- 支持拖动缩略图调整多图顺序。
- 新增粘贴图片上传能力。
- 强化 Prompt：要求模型逐图说明证据，并明确不要无依据默认“中等风险”。
- 默认模型改为 `qwen3-vl-flash`，更适合阿里云百炼免费额度和课程展示。
- 大幅升级界面：玻璃拟态面板、渐变背景、拖拽动画、上传动效、加载进度、报告仪表盘、分区卡片。
- 更新 README、API 文档和接入文档。

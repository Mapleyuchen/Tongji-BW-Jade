# 安全与合规说明

## 1. API Key 安全

API Key 只能放在 `server/.env` 或服务器环境变量中，不得写入前端代码、GitHub 仓库、截图或报告正文。

## 2. 上传安全

当前实现使用内存上传，不落盘保存图片。默认限制：

- 格式：JPG / PNG / WEBP
- 大小：8MB
- 单次请求：1 张图片

如后续需要保存用户图片，必须增加访问权限控制、删除机制和隐私说明。

## 3. 结果合规

系统不得宣传为“100% 鉴定真伪”。建议统一使用：

- AI 辅助鉴别
- 初步风险评估
- 疑似材质判断
- 专业送检建议

## 4. 免责声明推荐文本

本系统基于图像识别与视觉大模型进行玉石特征分析，鉴别结果仅供学习、展示和初步参考，不能作为交易、定价、维权或法律意义上的专业鉴定依据。对于高价值玉石，建议送至具备资质的珠宝玉石检测机构进行复检。

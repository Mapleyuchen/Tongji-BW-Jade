# 玉石学 - 古典鉴赏网站设计规范

## 1. Concept & Vision

**「玉韵」** —— 一个沉浸式的中国传统玉石鉴赏平台。网站以古典中式美学为核心，营造出博物馆般的典雅氛围，让访客仿佛置身于玉石珍藏馆中，感受千年玉文化的温润与厚重。界面设计追求「留白」与「意境」，以静谧的玉石绿为主调，金色点缀其间，展现玉石的高贵与灵韵。

## 2. Design Language

### 2.1 Aesthetic Direction
**新中式古典主义** —— 融合传统水墨意境与现代交互设计。以故宫红墙金瓦、苏州园林窗棂、敦煌壁画色彩为灵感，提炼出适合数字呈现的古典美学元素。

### 2.2 Color Palette
```
Primary (玉石绿):    #2D5A4A  - 主色调，代表翡翠的深邃
Secondary (暖玉白):   #F5F1E8  - 背景底色，如羊脂玉的温润  
Accent (古铜金):      #B8860B  - 点缀色，用于标题和装饰线条
Background (宣纸色):   #FDF8F0  - 主背景，如宣纸的质感
Text Primary:         #2C2C2C  - 正文深墨色
Text Secondary:       #6B5B4F  - 次要文字，茶褐色
Border (紫檀色):      #5C4033  - 边框与分隔线
```

### 2.3 Typography
- **标题字体**: "Noto Serif SC" (思源宋体) - 典雅庄重，用于所有标题
- **正文字体**: "Noto Sans SC" (思源黑体) - 清晰易读，用于正文
- **装饰字体**: "Ma Shan Zheng" (马善政楷书) - 用于诗词、引言等特殊场景
- **英文辅助**: "Cinzel" - 古典衬线体，用于装饰性英文

### 2.4 Spatial System
- 基础间距单位: 8px
- 内容最大宽度: 1200px
- 区块间距: 80px (大区块), 40px (小区块)
- 内边距: 32px (移动端 16px)

### 2.5 Motion Philosophy
- **入场动画**: 元素从透明渐入，配合轻微上移 (translateY: 20px → 0)，时长 600ms，缓动函数 cubic-bezier(0.22, 1, 0.36, 1)
- **悬停效果**: 玉石图片轻微放大 (scale: 1.02)，阴影加深，过渡 300ms
- **页面滚动**: 平滑滚动，导航栏在滚动后固定并添加背景模糊
- **装饰元素**: 云纹、祥云等图案有轻微的浮动动画，营造灵动氛围

### 2.6 Visual Assets
- **图标库**: 使用 Lucide Icons 搭配自定义 SVG 图标
- **装饰元素**: 
  - 传统云纹边框
  - 回字纹分隔线
  - 梅兰竹菊水墨图案
  - 玉石纹理背景
- **图片风格**: 高质量玉石摄影，浅色/白色背景，突出玉石质感

- **手机 (<768px)**: 单栏布局，汉堡菜单，底部导航

## 4. Features & Interactions

### 4.1 首页浏览 (index.html)
- 自动播放的玉石轮播展示
- 玉石分类快速入口（翡翠、和田玉、碧玉、独山玉等）
- 精选玉石展示网格

### 4.2 玉石鉴赏 (gallery.html)
- 玉石卡片网格展示
- 点击放大查看高清细节图
- 玉石基本信息展示（名称、产地、特征、寓意）
- 分类筛选（按种类、颜色、价格区间）
- 排序功能（默认/价格/名称）
- 分页显示

### 4.3 文化典藏 (culture.html)
- 玉之五德详解（仁义礼智信）
- 经典文献展示
- 玉文化历史时间线
- 玉言诗韵（经典诗词）

### 4.4 玉石创作 (creation.html)
- 形状选择（圆形、椭圆形、方形、水滴形、叶子形、竹节形）
- 玉石材质选择（翡翠绿、羊脂白、菠菜绿、紫罗兰、和田黄、墨玉）
- 雕刻图案选择（无、八卦、莲花、如意、寿字、福字、禄字、玉璧）
- 尺寸调整滑块
- 戒指尺寸选择
- Canvas 实时预览
- 设计保存与提交表单

### 4.5 专业鉴定 (appraisal.html)
- 图片上传功能（支持拖拽、多图上传）
- AI 鉴定流程展示（上传 → 分析 → 出报告）
- 鉴定结果详情（综合评分、等级、种类、产地、颜色、透明度、质地、雕工）
- 真伪鉴别（A货/B货/C货说明）
- 价值评估与价格区间
- 购买建议与保养指南
- 鉴定报告下载功能
- 翡翠ABC货知识科普

## 5. Component Inventory

### 5.1 Header 组件
- **默认状态**: 半透明背景，白色文字，云纹装饰
- **滚动状态**: 实色背景，阴影，分隔线
- **移动端**: 汉堡菜单按钮，滑出式导航

### 5.2 Navigation 组件
- **默认**: 文字无下划线
- **悬停**: 文字下方出现金色短横线动画
- **激活**: 金色下划线常驻

### 5.3 JadeCard 玉石卡片
- **默认**: 白色背景，细边框，阴影
- **悬停**: 阴影加深，图片放大，显示操作按钮
- **结构**: 图片区 + 标题 + 简介 + 标签

### 5.4 SectionTitle 区块标题
- **样式**: 中文主标题 + 英文副标题
- **装饰**: 左右金色短横线或云纹图案

### 5.5 CategoryTag 分类标签
- **默认**: 圆角矩形，边框样式
- **悬停**: 背景填充，文字变色
- **激活**: 实色背景

### 5.6 Lightbox 灯箱
- **结构**: 半透明黑色遮罩 + 居中图片
- **关闭按钮**: 右上角金色 X
- **导航**: 左右箭头切换

### 5.7 Footer 组件
- **结构**: 上分隔线 + 版权信息 + 装饰图案
- **风格**: 与整体古典风格统一

## 6. Technical Approach

### 6.1 技术栈
- **前端框架**: 纯 HTML5 + CSS3 + Vanilla JavaScript
- **无外部框架依赖**: 确保轻量级和快速加载
- **CSS 架构**: CSS Variables + CSS Grid + Flexbox
- **动画**: CSS Transitions + CSS Animations
